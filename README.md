# kurore_hp

**自作PC入門。Version 本音**のホームページ。

- <https://zipide.com>

## Development

```bash
npm install
npm start
```

## build

```bash
npm install
npm run build
```

`hp/docs/.vuepress/dist`にSSG後のHTMLファイル郡が出力される。

## release

**俺用**。

サーバーのwpコンテナにログイン

```bash
docker exec -it wp bash
```

```bash
cd 
./renew.sh
```
