CPU、CPUクーラー、メモリー、ストレージ、電源ユニットすべてが接続されます。

## 大きさについて

マザーボードには大きさの規格が存在します。主流なもので大きい順でATX、Micro-ATX、Mini-ITXの3種類です。PCケースが決まっているならば、PCケースに対応したマザーボードを選んでください。
